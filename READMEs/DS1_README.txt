== Installing ==
 * Extract DINPUT8.dll to the DARKSOULS.exe directory. 
   E.g. "C:\Program Files (x86)\Steam\steamapps\common\Dark Souls Prepare to Die Edition\DATA".

== What is this? ==
The No-Logo mod is a modification for Dark Souls 1 that completely removes the intro logo screens from the game.
These are the logos normally shown when first starting DS1.

== Supported Versions ==
v1.0.2.0
debug version

== Help ==

Q: The intro logo screens are still showing
A: Either the modified DINPUT8.dll is not in the correct directory or you are playing an unsupported version of the game.
